# Bibliotheque

Application de gestion de bibliotheque en Java.

## Fonctionnalites

- Ajouter un livre
- Afficher la collection
- Emprunter et retourner un livre
- Rechercher un livre par titre ou auteur

## Utilisation

Executez la classe `Main` pour lancer l'application.
1) javac Main.java
2) java Main
